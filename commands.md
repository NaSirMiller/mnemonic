# Comamnds Needed to Run Application

## Using Docker

## Start application (frontend and backend)

`docker compose up --build`

## Close application

1. ctrl + c in terminal
2. `docker compose down`

## Manual (Multi-shell)

### Frontend

1. `npm i`
2. `npm run dev`

### Backend

1. `npm i`
2. `npm run start:dev`
